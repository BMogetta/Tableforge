# Audio Assets Guide

This document lists reliable sources for **free sound effects** that can be used in this project.

The goal is to keep track of places where audio assets can be obtained safely, especially for **open source or non-commercial projects hosted on GitHub**.

---

## Freesound

<https://freesound.org>

Freesound is one of the largest community-driven libraries of sound effects. It contains thousands of sounds uploaded by users, including UI sounds, environmental audio, object interactions, and more.

Examples of useful sounds for board game projects:

* Dice rolls
* Card shuffling
* Button clicks
* Table interactions
* Ambient sounds

### License Notes

Freesound hosts content under several Creative Commons licenses:

* **CC0** → Free to use with no attribution required.
* **CC BY** → Free to use but attribution to the author is required.
* **CC BY-NC** → Non-commercial use only.

For this project, prefer using **CC0** or **CC BY** sounds.

If a sound requires attribution, include the author's credit in the repository documentation.

---

## Kenney Game Assets

<https://kenney.nl/assets>

Kenney provides high-quality assets specifically made for game development. These include UI sounds, game feedback sounds, and other useful effects.

Many indie developers use Kenney assets for prototypes and open source projects.

Examples of useful sounds:

* UI clicks
* Menu navigation sounds
* Coin / reward sounds
* Notification effects

### License Notes

Kenney assets are released under a **very permissive license**:

* Free for personal and commercial use
* No attribution required

Even though attribution is not required, it is still good practice to credit Kenney when possible.

---

## OpenGameArt

<https://opengameart.org>

OpenGameArt is a large open community repository of game development assets including audio, music, textures, sprites, and UI elements.

Many assets are released specifically for **open source or indie game projects**.

Examples of useful sounds:

* Retro game sound effects
* UI packs
* Environment sounds
* Interaction sounds

### License Notes

Assets on OpenGameArt may use different licenses:

* **CC0**
* **CC BY**
* **GPL**
* **OGA-BY License**

Always check the license listed on the asset page before using it.

---

## Zapsplat

<https://www.zapsplat.com>

Zapsplat is a large sound effects library containing thousands of audio files covering a wide range of categories.

Examples:

* Object sounds
* UI feedback
* Mechanical sounds
* Ambient environments

### License Notes

Free accounts can download sounds but **must provide attribution** to Zapsplat.

A paid subscription removes the attribution requirement.

---

## Attribution Example

If an asset requires attribution, include a section like this in the repository:

Audio assets used in this project:

* Sound: "Dice Roll"
  Author: John Doe
  Source: Freesound.org
  License: CC BY 4.0

---

## Recommended Usage Strategy

For this project:

* Prefer **CC0 assets** whenever possible.
* Use **CC BY assets** if attribution is simple to include.
* Avoid **CC BY-NC** if there is any chance the project might later become commercial.

Keeping track of licenses ensures the project remains safe to share publicly on GitHub.
